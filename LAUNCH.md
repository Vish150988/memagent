## Reddit Post (r/programming, r/MachineLearning)

**Title:** I built an open-source memory layer for AI coding agents (Claude Code, Codex, Cursor)

**Body:**
```
Every time I start a new Claude Code session, I have to re-explain my codebase. So I built Memagent — a local, open-source memory system that persists across sessions.

What it does:
- Stores decisions, preferences, errors, facts in SQLite
- Semantic search to find related memories by meaning
- Auto-generates CLAUDE.md so Claude Code auto-loads context
- Confidence decay: old memories fade unless reinforced

Tech stack: Python, Click, Rich, numpy (no OpenAI API, no cloud, no vector DB required)

GitHub: https://github.com/Vish150988/agentmemory

Would love feedback from anyone using AI agents for coding!
```

It's a local-first, cross-agent memory system:
- SQLite storage (private, fast, portable)
- Semantic search via TF-IDF + cosine similarity (numpy only)

- Memory confidence decay + reinforcement
